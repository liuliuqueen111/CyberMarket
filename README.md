# 🌐 Cyber Market - 游戏装备商店

Cyber Market 是一个充满科技感的游戏装备商店，为玩家提供沉浸式的装备采购体验。商店集成了从装备浏览、详情查看到购物车管理一系列商店采购功能。

## ✨ 核心功能

我们提供了丰富的装备浏览体验，支持智能搜索功能，用户可以通过装备名称快速定位心仪物品。使用多维度筛选可以让玩家根据装备类型和游戏需求进行精准筛选，找到所需的战斗装备、潜行工具和医疗植入体。

购物车系统采用侧边抽屉设计，玩家可以实时调整装备数量，查看总价，并享受数据持久化的便利，即使关闭浏览器也不会丢失购物记录。每个装备都配备了详细的参数说明、视觉效果预览和库存状态显示。

## 🎨 交互体验

商店的界面设计融入了大量赛博朋克元素，包括霓虹发光效果、3D倾斜卡片动画和流畅的页面切换。商品卡片在鼠标悬停时会产生动态倾斜效果，营造出科技感十足的视觉体验。响应式设计确保了在不同设备上的完美适配，从桌面电脑到移动设备都能提供一致的高质量体验。

## 🏗️ 技术架构

项目基于 React 18 和 TypeScript 构建，采用 Vite 作为开发和构建工具，确保快速的开发体验和优化的生产性能。Tailwind CSS 提供了强大的样式系统，支持复杂的视觉效果和响应式布局。Zustand 作为轻量级状态管理解决方案，负责处理购物车、筛选条件等应用状态。Lucide React 图标库为装备提供了丰富的视觉标识。

## 🚀 快速开始

克隆项目到本地后，安装依赖并启动开发服务器：

```bash
npm install
npm run dev
```

项目将在 http://localhost:5173 上运行，你可以立即开始浏览和测试装备商店的各项功能。

## 📱 技术特性

商店支持完整的电商工作流，包括商品展示、详情页、购物车和搜索过滤。所有数据都经过精心设计，包含50种不同的游戏装备，每件装备都有独特的图标、详细描述和参数信息。持久化存储确保了用户的购物体验不会因页面刷新而中断。

这个项目不仅是一个功能完整的装备商店，更是一个展示现代前端技术栈能力的优秀案例，完美结合了美学设计和技术实现。
